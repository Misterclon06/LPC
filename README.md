# Estructuras de Datos en C

[![Language](https://img.shields.io/badge/language-C-blue.svg)](https://en.wikipedia.org/wiki/C_(programming_language))
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Una biblioteca para estructuras de datos lineales en C, usando operaciones basicas de TDA-LPC vistas en las asignatura algoritmos I. Implementa listas enlazadas, pilas y colas con una API intuitiva y fácil de usar.

## Características

- **Sin dependencias externas**: Solo utiliza la biblioteca estándar de C
- **Manejo de memoria**: Gestión de memoria dinámica
- **Operaciones avanzadas**: Incluye ordenamiento, búsqueda, transformaciones, etc.
- **Bien documentada**: Código comentado y fácil de entender

## 📚 Estructuras Incluidas

### Lista Enlazada
- Lista simplemente enlazada con operaciones completas
- Inserción, eliminación, búsqueda y ordenamiento
- Operaciones de slicing, concatenación y copia

### Pila (Stack)
- Implementación LIFO (Last In, First Out)
- Operaciones básicas y avanzadas
- Funciones de utilidad para arrays

### Cola (Queue)
- Implementación FIFO (First In, First Out)
- Operaciones de rotación y transformación

## 🔧 Requisitos

- Compilador de C (GCC, Clang, MSVC)
- Make (opcional, para compilación)
